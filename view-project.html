News Collecting App using Angular and Firebase

Breakdown of the idea

I find it difficult in todays socity to know if the articals im reading are fake news or politically anged news. Also because of social media news travals really fast and gets twisted along the way. A sulotion to the readers problem would be to create a application that collects articals from diffrent news sources so that the readers can get a bigger picture of the information they are taking in.


The Latest is a News collecting application built with Angular using three diffrent news api end-points to collect articals from differnt news sites.

For every artical there is a implemented comment section and rating system for users to write their thoughts and to rate the artical.

I used Firebase-store as my data storage for the comments and the ratings so that when the page is reload the comments will still be save and displayed on the application.
