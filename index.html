News Collecting App using Angular and Firebase

Breakdown of the idea

I find it difficult in todays society to know if the articles I'm reading are fake news or politically angled news. Also because of how news travels really fast because of social media and gets twisted along the way. A solution to the problem would be to create a application that collects articles from different news sources so that it's easier to create a bigger picture of the information thats given.


The Latest is a News collecting application built with Angular using three different news api end-points to collect articles from different news sites.

For every article there is a comment and rating section system for users to write their thoughts and to give the article a raiting.

I used Firebase cloud-store as my data storage for the comments and the ratings so that when the page is reload the comments will still be save and displayed on the application.
